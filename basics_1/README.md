# Réseau Informatique - Holberton School

Ce dépôt contient les projets et exercices dédiés à l'apprentissage des concepts fondamentaux du réseau informatique au sein du cursus Holberton School. Les scripts sont développés en Bash et configurés pour les environnements de serveurs Ubuntu.

## 📂 Structure du dépôt

Le projet est divisé en plusieurs répertoires, chacun traitant de notions spécifiques :

```text
.
├── basics_0/
│   ├── 0-OSI_model               # Réponses au quiz sur les bases du modèle OSI
│   ├── 4-TCP_and_UDP_ports       # Script affichant les ports TCP/UDP en cours d'écoute
│   └── 5-is_the_host_on_the_network # Script utilisant ICMP (ping) pour tester une IP
└── basics_1/
    └── 0-change_your_home_IP     # Script modifiant /etc/hosts pour dévier des résolutions DNS
